# Phase 1: 기반 구축 및 TipTap 에디터 설정

**목표**: Tauri 환경 세팅 및 TipTap 에디터를 Notion 스타일로 구성하여 3패널 레이아웃 완성.

## 완료

- [x] Tauri + React + TypeScript + Vite 프로젝트 생성
- [x] 필수 라이브러리 설치: zustand, tailwindcss, langchain
- [x] TipTap 패키지 설치: @tiptap/react, @tiptap/starter-kit, @tiptap/extension-link, @tiptap/extension-placeholder
- [x] 전역 테마 및 Pretendard 폰트 설정
- [x] TipTap 기본 에디터 컴포넌트 생성
- [x] Source/Target TipTap 에디터 구현
- [x] 지원 포맷 Extension 설정 (StarterKit, Heading, Lists, Marks, Blockquote, Link, Placeholder)
- [x] Notion 스타일 CSS 적용
- [x] 3패널 레이아웃 및 Focus Mode 토글
