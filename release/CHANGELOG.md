# Changelog

모든 주요 변경사항을 기록합니다.

## [1.1.0] - 2025-01-21

### 추가
- 번역 검수 기능 (오역/누락/일관성 검출)
- 에디터 내 검색/치환 (Cmd+F, Cmd+H)
- 플로팅 채팅 패널 (드래그/리사이즈 가능)
- 채팅 패널 고정(Pin) 기능
- Multi-Provider AI 지원 (OpenAI + Anthropic 동시 사용)

### 개선
- 대용량 문서 번역 성능 최적화
- 채팅 응답 스트리밍 안정성 향상
- 메모리 사용량 개선

### 수정
- 프로젝트 전환 시 채팅 내역 혼합 버그
- 에디터 하이라이트가 사라지는 문제
- 한글 입력 시 Enter 키 오동작
