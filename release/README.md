# OddEyes.ai

> "AI를 동료로, 번역을 예술로"

전문 번역가를 위한 AI 기반 번역 워크스테이션

<!-- ![OddEyes.ai Screenshot](assets/screenshot.png) -->

## 특징

- **Notion 스타일 에디터**: 원문/번역문 나란히 편집
- **AI 번역 지원**: OpenAI, Anthropic 모델 선택 가능
- **번역 검수**: 오역, 누락, 일관성 자동 검출
- **용어집 관리**: 프로젝트별 용어 통일
- **외부 연동**: Confluence, Notion, 웹 검색

## 설치 방법

### macOS

1. DMG 파일 다운로드
2. 더블클릭하여 마운트
3. `OddEyes.ai`를 `Applications` 폴더로 드래그
4. 처음 실행 시 "확인되지 않은 개발자" 경고가 나타나면:
   - **우클릭 → 열기** 선택
   - 또는 터미널에서: `xattr -cr /Applications/OddEyes.ai.app`

## 시작하기

1. 앱 실행
2. **Settings** (우측 사이드바) → API Key 입력
   - OpenAI API Key 또는
   - Anthropic API Key
3. 새 프로젝트 생성
4. 원문 입력 → 번역 버튼 클릭

## 요구사항

- macOS 12 Monterey 이상
- OpenAI 또는 Anthropic API Key
