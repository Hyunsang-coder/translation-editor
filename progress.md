# Progress

## 최근 변경 (2025-12)

- Source/Target을 TipTap 문서 편집기로 통일하고, Source도 붙여넣기/정리 목적으로 편집 가능하게 조정
- 선택 텍스트 → 채팅 입력창 복사 UX를 “우클릭” 기반으로 단순화
- “문서 전체 번역(Preview → Apply)” 워크플로우를 PRD/TRD에 명시 (최근 채팅 10개 컨텍스트 포함)
- 채팅 입력창(Composer) UI를 첨부 이미지 스타일로 교체(우하단 화살표 Send, 좌하단 `+` 메뉴)
- `+` 메뉴에 웹 검색 토글을 추가하고, ON일 때만 Brave/OpenAI web search가 동작하도록 게이트 적용
- 프로젝트 첨부에 이미지(png/jpg/jpeg/webp/gif)를 추가하고, 채팅 호출 시 멀티모달(vision) 입력으로 전달
